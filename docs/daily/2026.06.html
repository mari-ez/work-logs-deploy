      1. 직군 역할 한정: 가치 노출 목록 + 설명 콘텐츠 강조에만 영향, alt 생성 정책에는 미반영 (같은 건물+같은 가치 = 동일 alt 결정론 원칙)

      3. 가치 = (충진 우선순위, 정지조건, 보정계수) 3요소 세트로 매핑
    5. 집단에너지 처리 확정: 가치가 아닌 조건 플래그로 분리(alt 저장 스키마 별도 필드), MVP는 아파트 한정·배타 선택 UI·사용자 직접 선택(공시 정보로 자동판별 불가), 추후 공식 데이터 확보 시 자동판별·조합 생성 확장 구조
